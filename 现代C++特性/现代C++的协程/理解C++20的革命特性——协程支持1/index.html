    вҖӢ еҚҸзЁӢжңүдёӨзұ»е®һзҺ°жҖқи·Ҝпјҡжңүж ҲеҚҸзЁӢпјҲstackfulпјүдјҡеҲҮжҚўе®Ңж•ҙзҡ„жү§иЎҢж ҲпјӣиҖҢC++20 зҡ„еҚҸзЁӢеұһдәҺ"ж— ж ҲпјҲstacklessпјү"иҢғејҸвҖ”вҖ”зј–иҜ‘еҷЁдјҡжҠҠеңЁжҢӮиө·зӮ№йңҖиҰҒдҝқз•ҷзҡ„еұҖйғЁеҸҳйҮҸе’ҢзҠ¶жҖҒе°ҒиЈ…еҲ°дёҖдёӘ еҚҸзЁӢеё§пјҲcoroutine frameпјүдёӯгҖӮжҢӮиө·ж—¶дҝқеӯҳиҜҘеҚҸзЁӢеё§е№¶иҝ”еӣһпјҢжҒўеӨҚж—¶д»Һеё§йҮҢжҒўеӨҚзҠ¶жҖҒз»§з»ӯжү§иЎҢгҖӮеӣ дёәдёҚйңҖиҰҒеҲҮжҚўж“ҚдҪңзі»з»ҹж ҲпјҢд№ҹйҖҡеёёдёҚйңҖиҰҒйў‘з№Ғиҝӣе…ҘеҶ…ж ёжҖҒпјҢеҜ№дәҺжһҒз«Ҝзҡ„е№¶еҸ‘еңәжҷҜпјҢиҝҷзҺ©ж„Ҹжҳҫз„¶жҜ”иҝӣзЁӢ/зәҝзЁӢзҡ„еҲҮжҚўиҰҒејәзҡ„еӨӘеӨҡеӨӘеӨҡгҖӮ

    жҲ‘д»¬дҪҝз”ЁеҚҸзЁӢйҖҡеёёжңүдёүеӨ§зҗҶз”ұпјҡ

    • й«ҳе№¶еҸ‘гҖҒдҪҺејҖй”ҖпјҡзӣёжҜ”зәҝзЁӢпјҢеҚҸзЁӢзҡ„еҲӣе»әдёҺеҲҮжҚўд»Јд»·жӣҙдҪҺпјҢйҖӮеҗҲеӨ§йҮҸ I/O еҜҶйӣҶеһӢе№¶еҸ‘д»»еҠЎгҖӮ
    • жӣҙзҒөжҙ»зҡ„жҺ§еҲ¶жөҒиЎЁиҫҫпјҡеҚҸзЁӢеӨ©з”ҹйҖӮеҗҲе®һзҺ°з”ҹжҲҗеҷЁгҖҒжөҒж°ҙзәҝгҖҒжғ°жҖ§и®Ўз®—дёҺејӮжӯҘд»»еҠЎй“ҫзӯүжЁЎејҸгҖӮ

    вҖӢ жҲ‘д»¬иҝҷйҮҢжҳҜC++зҡ„еҚҡе®ўпјҢйҒҝе…ҚдёҚдәҶи®Ёи®әC++зҡ„еҚҸзЁӢж”ҜжҢҒгҖӮдҪҶеҫҲйҒ—жҶҫпјҢз¬”иҖ…еҝ…йЎ»ејәи°ғзҡ„жҳҜвҖ”вҖ”C++20зҡ„еҚҸзЁӢжҺҘеҸЈжҳҜеңЁжҜ”иҫғйҡҫеҶҷгҖӮз¬”иҖ…йҖӣдәҶдәӣи®әеқӣпјҢеҢ…жӢ¬зңӢеҲ°е…¶д»–еҗҢеҝ—еҜ№C++20зҡ„еҚҸзЁӢзҡ„д»Ӣз»ҚпјҢдёҚеҫ—дёҚжүҝи®ӨвҖ”вҖ”иҝҷдёҖеҘ—жҺҘеҸЈеҰӮжһңжҲ‘д»¬дёҚзҗҶи§ЈеҚҸзЁӢпјҢе®һеңЁжҳҜйҡҫд»ҘзҗҶи§ЈпјҲдёәжӯӨжҲ‘жҢЈжүҺдәҶеҘҪдёҖдјҡпјүгҖӮеӣ жӯӨпјҢз¬”иҖ…йқһеёёе»әи®®еҗ„дҪҚзңӢжӯӨеҚҡе®ўзҡ„ж—¶еҖҷпјҢеӨҡз»ғд№ д»Јз ҒпјҢжү“дёҖжү“ж—Ҙеҝ—гҖӮиҝҷж ·жңүеҠ©дәҺдҪ зҗҶи§ЈвҖ”вҖ”C++зҡ„еҚҸзЁӢеҲ°еә•еңЁеҒҡд»Җд№ҲгҖӮ

    вҖӢ дёәдәҶеұ•ејҖиҜҙжҳҺдёҠйқўзҡ„еҶ…е®№пјҢз¬”иҖ…еҶіе®ҡйҮҚж–°ж•ҙзҗҶдёҖдёӢcppreferenceеҜ№дәҺеҚҸзЁӢзҡ„д»Ӣз»Қ

    жҲ‘зҹҘйҒ“жңүдёҖдәӣжңӢеҸӢиҝҳжІЎзңӢд»Җд№ҲжҳҜC++дёӯзҡ„еҚҸзЁӢпјҢжӮЁеҸҜд»ҘиҮӘиЎҢе…ҲзңӢзңӢcppreferenceеҜ№иҝҷдёӘжҺҘеҸЈзҡ„и®Іиҝ°пјҢз¬”иҖ…з¬¬дёҖж¬ЎзңӢеҲ°дёҖеҚҠе°ұе…іжҺүеҶҷеҲ«зҡ„еҺ»дәҶпјҢе®һеңЁжңүдәӣйҡҫжҮӮпјҒрҹ‘үеҚҸзЁӢ (C++20) - cppreference.cn - C++еҸӮиҖғжүӢеҶҢ

    вҖӢ ж•ҙзҗҶдёӢжқҘвҖ”вҖ”жҲ‘д»¬йңҖиҰҒдәҶи§ЈиҝҷдәӣеҶ…е®№пјҢжӮЁи®°еңЁжүӢеӨҙеҒҡдёҖдёӘз¬”и®°гҖӮжҲ–иҖ…дҪ дёҚжғізңӢзҡ„иҜқпјҢеҸҜд»Ҙи·іеҲ°дёӢдёҖйғЁеҲҶзңӢдёҖдёӢдҫӢеӯҗпјҢдҪ жү«дёҖзңје°ұзҹҘйҒ“еӨ§жҰӮжҲ‘д»¬йңҖиҰҒеҰӮдҪ•дҪҝз”ЁC++20ж”ҜжҢҒзҡ„еҚҸзЁӢдәҶ

    • зј–иҜ‘еҷЁжҸҗдҫӣзҡ„дёүдёӘжү©еұ•е…ій”®еӯ—йңҖиҰҒжҲ‘д»¬е…ҲзҹҘйҒ“пјҡ

    • co_awaitпјҡиҝҷдёӘе…ій”®еӯ—з”ЁдәҺжҠҠеҚҸзЁӢжҢӮиө·жқҘпјҢзӣҙеҲ°жҲ‘д»¬и°ғз”ЁдәҶжҒўеӨҚжңәеҲ¶жҠҠе®ғж”ҫдёӢжқҘпјҒпјҢйңҖиҰҒиҜҙжҳҺзҡ„жҳҜвҖ”вҖ”е’ұд»¬зҡ„co_awaitзҡ„еҗҺйқўйңҖиҰҒи·ҹдёҖдёӘиЎЁиҫҫејҸгҖӮиҝҷдёӘиЎЁиҫҫејҸеҫҖеҫҖжҳҜдёҖдёӘж”ҜжҢҒиӢҘе№ІC++зәҰе®ҡеҚҸзЁӢжҺҘеҸЈзҡ„еҜ№иұЎпјҲиҮіе°‘з¬”иҖ…зӣ®еүҚиҝҷж ·дҪҝз”ЁпјҢеҗ„дҪҚC++еҚҸзЁӢеӨ§и·ҢиҠұжӢӣеҫҲеӨҡпјҢзңӢзқҖе®һеңЁиҙ№и§ЈйҡҫжҮӮпјҢжүҖд»ҘзҙўжҖ§е…Ҳиҝҷж ·иҜҙпјҢдҫҝдәҺеҲқзә§иҜ»иҖ…зҡ„зҗҶи§ЈпјүгҖӮдәәиҜқе°ұжҳҜпјҢзӯүеҫ…зҡ„дёңиҘҝдёӯиҰҒе®һзҺ°з»ҷе®ҡзӯҫеҗҚзҡ„еҮҪж•°пјҢдёҚе®һзҺ°зј–иҜ‘еҷЁе°ұдјҡе‘ҠиҜүдҪ жҺҘеҸЈзјәеӨұпјҒ

    • co_yieldпјҡз”ЁдәҺжҡӮеҒңжү§иЎҢе№¶иҝ”еӣһдёҖдёӘеҖјгҖӮе•Ҙж„ҸжҖқе‘ўпјҢиҝҷдёӘж—¶еҖҷж”ҫеңЁе’ұд»¬зҡ„еҚҸзЁӢеҮҪж•°йҮҢпјҢд»–е°ұдјҡиҝ”еӣһco_yieldдҝ®йҘ°зҡ„иЎЁиҫҫејҸеӯҗзҡ„еҖјпјҢиҝҷдёӘеҖјйңҖиҰҒеҲ©з”ЁдёҖдёӘжҺҘеҸЈиҝ”еӣһеӣһеҺ»гҖӮе…·дҪ“жҖҺд№Ҳз”ЁеҲ«зқҖжҖҘпјҢжҲ‘д»¬еҗҺйқўдјҡи®І

    • co_returnпјҡз”ЁдәҺе®ҢжҲҗжү§иЎҢе№¶иҝ”еӣһдёҖдёӘеҖјпјҢиҝҷдёӘж—¶еҖҷжҲ‘д»¬еҶҷдёӢдёҖдёӘco_returnпјҢиҝҷдёӘеҚҸзЁӢеҮҪж•°е°ұз»“жқҹдәҶпјҢеҮҶеӨҮй”ҖжҜҒжҲ‘д»¬зҡ„еҚҸзЁӢз»“жһ„дҪ“гҖӮ

    • иҝҳжңүдёҖйғЁеҲҶжҳҜдёҖдёӘеҚҸзЁӢеҮҪж•°йңҖиҰҒиҝ”еӣһзҡ„дёҖдёӘз»“жһ„дҪ“пјҲеҚҸзЁӢиҝ”еӣһзұ»еһӢпјүпјҢиҝҷдёӘз»“жһ„дҪ“иў«з”ЁжқҘз»ҷеҚҸзЁӢжЎҶжһ¶жҸҗдҫӣдёҖе®ҡзҡ„и°ғеәҰдҝЎжҒҜгҖӮе®һйҷ…дёҠпјҢе’ұд»¬зҡ„зҺ°д»ЈC++йғҪжҳҜдҪҝз”Ёзҡ„жҺҘеҸЈжқҘиЎЁзӨәиғҪдёҚиғҪж”ҜжҢҒеҚҸзЁӢпјҢжүҖд»ҘпјҢжҲ‘д»¬йңҖиҰҒеҒҡзҡ„жҳҜеЈ°жҳҺдёҖдёӘеҜ№иұЎзұ»еһӢпјҢд»–еҝ…йЎ»еҶ…еөҢpromise_type,жіЁж„Ҹе°ұжҳҜиҝҷдёӘеҗҚз§°пјҢеҸҳдёҚдәҶпјҒ

    // coroutineдёӯ
#if __cpp_concepts
    requires requires { typename _Result::promise_type; }
    struct __coroutine_traits_impl<_Result, void>
#else
    struct __coroutine_traits_impl<_Result,
       __void_t<typename _Result::promise_type>>
#endif
    {
      using promise_type = typename _Result::promise_type;
    };

    дёӢдёҖжӯҘпјҢе°ұжҳҜеЈ°жҳҺе’Ңе®һзҺ°иҝҷдёӘpromise_typeдёӯеҝ…йЎ»иҰҒеӯҳеңЁзҡ„жҺҘеҸЈпјҢиҝҷе°ұжҳҜжҲ‘д»¬иҰҒе®һзҺ°зҡ„вҖ”вҖ”

    жҺҘеҸЈ (еҮҪж•°) дҪңз”Ё иҝ”еӣһзұ»еһӢиҰҒжұӮ
    1. get_return_object() иҺ·еҸ–иҝ”еӣһеҜ№иұЎпјҡеҚҸзЁӢеҮҪж•°иў«и°ғз”Ёж—¶з¬¬дёҖдёӘжү§иЎҢзҡ„еҮҪж•°гҖӮе®ғиҙҹиҙЈеҲӣе»әе№¶иҝ”еӣһи°ғз”ЁиҖ…пјҲеӨ–йғЁдё–з•Ңпјүз”ЁдәҺж“ҚдҪңеҚҸзЁӢзҡ„иҝ”еӣһеҜ№иұЎпјҲеҰӮжӮЁзҡ„ GeneratorпјүгҖӮ еҝ…йЎ»иҝ”еӣһеҚҸзЁӢеҮҪж•°зҡ„иҝ”еӣһзұ»еһӢпјҲжҲ–еҸҜиҪ¬жҚўдёәиҜҘзұ»еһӢпјүгҖӮ
    2. initial_suspend() еҲқе§ӢжҡӮеҒңзӮ№пјҡеҶіе®ҡеҚҸзЁӢеңЁеҲӣе»әж—¶жҳҜз«ӢеҚіжү§иЎҢиҝҳжҳҜжҡӮеҒңгҖӮ еҝ…йЎ»иҝ”еӣһдёҖдёӘ Awaitable еҜ№иұЎпјҲеҰӮ std::suspend_always жҲ– std::suspend_neverпјүгҖӮ
    3. final_suspend() жңҖз»ҲжҡӮеҒңзӮ№пјҡеҶіе®ҡеҚҸзЁӢеңЁжү§иЎҢе®ҢжҜ•пјҲco_return жҲ–еҮҪж•°дҪ“з»“жқҹпјүеҗҺжҳҜз«ӢеҚій”ҖжҜҒиҝҳжҳҜжҡӮеҒңгҖӮ еҝ…йЎ»иҝ”еӣһдёҖдёӘ Awaitable еҜ№иұЎгҖӮ
    4. return_void() жҲ– return_value(V) иҝ”еӣһеҖјеӨ„зҗҶпјҡз”ЁдәҺеӨ„зҗҶеҚҸзЁӢзҡ„з»Ҳз»“еҖјжҲ–з»Ҳз»“зҠ¶жҖҒгҖӮ еҰӮжһңеҚҸзЁӢеҮҪж•°иҝ”еӣһ void (дҫӢеҰӮ Generator з»ҸеёёеҰӮжӯӨ)пјҢеҝ…йЎ»жҸҗдҫӣ return_void()гҖӮеҰӮжһңеҚҸзЁӢдҪҝз”Ё co_return V; иҝ”еӣһдёҖдёӘеҖјпјҢеҲҷеҝ…йЎ»жҸҗдҫӣ return_value(V)гҖӮдёӨиҖ…дәҢйҖүдёҖгҖӮ
    5. unhandled_exception() ејӮеёёеӨ„зҗҶпјҡеҪ“еҚҸзЁӢеҶ…йғЁеҸ‘з”ҹжңӘжҚ•иҺ·зҡ„ејӮеёёж—¶иў«и°ғз”ЁгҖӮ еҝ…йЎ»иҝ”еӣһ voidгҖӮ

    еҪ“з„¶пјҢиҝҳиҰҒеҖјеҫ—дёҖжҸҗзҡ„жҳҜпјҢеҰӮжһңдҪ зҡ„еҚҸзЁӢеҮҪж•°дёӯпјҢз”ЁеҲ°дәҶco_yieldе…ій”®еӯ—пјҢдҪ иҝҳйңҖиҰҒйўқеӨ–зҡ„жҗһе®ҡдёҖдёӘеҮҪж•°

    жҺҘеҸЈ (еҮҪж•°) дҪңз”Ё иҝ”еӣһзұ»еһӢиҰҒжұӮ
