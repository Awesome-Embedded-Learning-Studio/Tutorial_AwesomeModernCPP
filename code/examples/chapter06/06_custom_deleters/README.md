# Chapter 6.6: 自定义删除器（Custom Deleter）

本目录包含自定义删除器的示例，展示如何管理非标准资源。

## 文件说明

| 文件 | 描述 |
|------|------|
| `file_handle.cpp` | FILE* 和文件描述符的自定义删除器 |
| `memory_pool.cpp` | 内存池与智能指针配合使用 |

## 关键概念

- **函数指针删除器**: 会增加 unique_ptr 大小
- **无状态函数对象**: 通过 EBO 实现零开销
- **有状态删除器**: 包含上下文信息，会增加大小
- **自定义 RAII 包装器**: 有时比智能指针更合适

## 删除器类型选择

| 类型 | 大小开销 | 适用场景 |
|------|----------|----------|
| 无状态函数对象 | 零 | 默认选择 |
| 函数指针 | 指针大小 | 需要运行时切换 |
| 有状态删除器 | 取决于状态 | 需要上下文 |
| 自定义 RAII 类 | 最小 | 资源不是指针 |

## 编译运行

```bash
mkdir build && cd build
cmake ..
make
./file_handle
./memory_pool
```
