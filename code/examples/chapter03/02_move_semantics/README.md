# Chapter 3.2 - 移动语义示例

本目录包含《移动语义》章节的代码示例。

## 说明

移动语义允许资源所有权的转移，避免昂贵的拷贝操作。这在嵌入式系统中管理大型缓冲区和硬件资源时尤为重要。

## 编译方法

```bash
mkdir build && cd build
cmake ..
make
./move_semantics_example
```

## 关键点

1. **移动的本质**：转移资源所有权，而非拷贝数据
2. **硬件资源管理**：禁用拷贝、启用移动确保独占所有权
3. **动态资源**：移动构造实现零拷贝转移
4. **容器操作**：push_back/emplace_back 使用移动语义
5. **完美转发**：T&& 在模板中保留值类别

## 何时使用移动语义

- 管理硬件资源的 RAII 类
- 持有大型缓冲区的数据结构
- 容器元素类型
- 工厂函数和构建器模式

## 环境要求

- C++20 或更高
- GCC 10+ 或 Clang 10+
