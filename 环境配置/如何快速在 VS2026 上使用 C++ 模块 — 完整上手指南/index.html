    еҲӣе»әдёҖдёӘе°Ҹе·ҘзЁӢ vs2026-modules-demo/пјҢеҢ…еҗ«дёӨдёӘж–Үд»¶пјҡ

    math.ixxпјҲжЁЎеқ—жҺҘеҸЈеҚ•е…ғпјүпјҡ

    ```cpp export module math;

    export int add(int a, int b) { return a + b; }

    export struct Point { int x, y; }; ```text

    main.cppпјҲдҪҝз”ЁжЁЎеқ—пјүпјҡ

    ```cpp import std; import math;

    int main() { std::print("Add Result: {}", add(1, 2)); Point p{ 1,2 }; std::print("Point p ({}, {})\n", p.x, p.y); return 0; } ```text

    иҜҙжҳҺпјҡMSVC зӨҫеҢәдёӯ .ixx жҳҜеёёи§Ғзҡ„жЁЎеқ—жҺҘеҸЈжү©еұ•еҗҚпјӣдҪ д№ҹеҸҜд»ҘдҪҝз”Ё .cppm зӯүпјҢдҪҶ IDE/е·Ҙе…·й“ҫеҜ№жү©еұ•еҗҚзҡ„й»ҳи®ӨиҜҶеҲ«еҸҜиғҪдёҚеҗҢгҖӮ

    4. жһ„е»әе№¶иҝҗиЎҢпјҡIDE дјҡиҮӘеҠЁеҜ№жЁЎеқ—жәҗиҝӣиЎҢжү«жҸҸгҖҒз”ҹжҲҗ BMIгҖҒе№¶жӯЈзЎ®и®ҫзҪ®зј–иҜ‘дёҺй“ҫжҺҘж¬ЎеәҸпјӣдҪ йҖҡеёёдёҚйңҖиҰҒжүӢеҠЁжҢҮе®ҡ .objгҖӮеҰӮжһңжЁЎеқ—й—ҙдҫқиө–еӨҚжқӮпјҲи·ЁйЎ№зӣ®пјүпјҢеҸҜд»ҘдҪҝз”ЁйЎ№зӣ®еј•з”ЁжҲ–еңЁ Project Properties дёӯй…ҚзҪ® Module ReferencesгҖӮ
