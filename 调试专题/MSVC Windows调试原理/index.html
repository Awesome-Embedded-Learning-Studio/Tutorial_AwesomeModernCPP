    жҲ‘д»¬дҪңдёәдё»еҠЁзҡ„дёҖж–№пјҢе°ұжҳҜиҙҹиҙЈзӮ№еҮ»Visual Studio IDE (The Shell)жҸҗдҫӣзҡ„GUIз•ҢйқўдёӢиҫҫжҢҮд»ӨпјҢдҪҶжҳҜжҲ‘иҰҒиҜҙдёҖеҸҘиҜқпјҢVSдёҚиҙҹиҙЈе®һйҷ…зҡ„и°ғиҜ•йҖ»иҫ‘пјҢе®ғеҸӘиҙҹиҙЈжҳҫзӨәпјҢе®ғе°Ҷз”ЁжҲ·зҡ„зӮ№еҮ»ж“ҚдҪңпјҲеҰӮ F10пјүиҪ¬жҚўдёәеҸ‘йҖҒз»ҷи°ғиҜ•еј•ж“Һзҡ„е‘Ҫд»ӨгҖӮ

    жҜ”иҫғйҮҚиҰҒзҡ„дёҖдёӘз»„д»¶жҳҜи°ғиҜ•еј•ж“Һ (Debug Engine, DE)пјҢе®ғиҙҹиҙЈи§ЈжһҗеӨҚжқӮзҡ„ C++ иЎЁиҫҫејҸпјҲжҜ”еҰӮ vec[0].m_dataпјүпјҢиҙҹиҙЈиҜ»еҸ– PDB з¬ҰеҸ·ж–Үд»¶пјҢе°Ҷең°еқҖ 0x00401234 зҝ»иҜ‘дёә main.cpp:20пјҲжңүзӮ№еғҸgnuе·Ҙе…·й“ҫзҡ„addr2lineдәҶпјү

    msvsmon.exe (Remote Debugging Monitor)жҳҜжү§иЎҢиҖ… / д»ЈзҗҶдәә / йҡ”зҰ»еұӮпјҢжҲ‘д»¬зҹҘйҒ“и°ғиҜ•зҡ„ж—¶еҖҷжҳҜжҲ‘д»¬IDEиҝӣзЁӢжӢүиө·жқҘиҝҷдёӘи°ғиҜ•иҝӣзЁӢпјҢmsvsmonзҡ„дҪңз”Ёе°ұжҳҜеҰӮжһңзӣ®ж ҮзЁӢеәҸеҙ©жәғжҲ–жҢӮиө·пјҢдёҚдјҡеҜјиҮҙ VS IDE еҙ©жәғпјӣеҗҢж—¶msvsmon иҙҹиҙЈеңЁ IDE е’Ңзӣ®ж ҮиҝӣзЁӢд№Ӣй—ҙдј йҖ’ж•°жҚ®гҖӮе®ғжҳҜе®һйҷ…и°ғз”Ё Windows API еҺ»жҺ§еҲ¶зӣ®ж ҮиҝӣзЁӢзҡ„йӮЈдёӘ"дәә"гҖӮ

    е…ідәҺWindowsеҶ…ж ёзҡ„дҪңз”ЁиҝҷйҮҢе’ұд»¬е°ұжҺ иҝҮдәҶпјҢж— йқһе°ұжҳҜжҸҗдҫӣи°ғиҜ•зӣёе…ізҡ„System APIиҖҢе·ІгҖӮ

    з¬¬дёҖйҳ¶ж®өпјҡе»әз«ӢиҝһжҺҘ (Establishment)¶

    еңЁиҝӣиЎҢиҝңзЁӢи°ғиҜ•ж—¶пјҢдёҖеҲҮе§ӢдәҺи°ғиҜ•еҷЁпјҲDebuggerпјүдёҺе®ҝдё»зі»з»ҹзҡ„дәӨдә’гҖӮе…·дҪ“иҖҢиЁҖпјҢVisual Studio дјҡйҖҡиҝҮиҝңзЁӢи°ғиҜ•зӣ‘и§ҶеҷЁпјҲmsvsmon.exeпјүеҸ‘иө·иҜ·жұӮпјҢи°ғз”Ёе…ій”®зҡ„ Win32 API вҖ”вҖ” CreateProcessгҖӮеңЁи°ғз”ЁиҝҮзЁӢдёӯпјҢзі»з»ҹдјҡдј е…ҘдёҖдёӘиҮіе…ійҮҚиҰҒзҡ„ж Үеҝ—дҪҚ DEBUG_ONLY_THIS_PROCESSпјҲжҲ– DEBUG_PROCESSпјүгҖӮиҝҷдёӘж Үеҝ—дҪҚдёҚд»…жҳҜеҗҜеҠЁжҢҮд»ӨпјҢжӣҙжҳҜеҗ‘ж“ҚдҪңзі»з»ҹеҸ‘еҮәзҡ„"жҺҘз®ЎеЈ°жҳҺ"пјҢж Үеҝ—зқҖзӣ®ж ҮиҝӣзЁӢд»ҺиҜһз”ҹд№ӢеҲқе°ұеӨ„дәҺеҸ—жҺ§зҠ¶жҖҒгҖӮ

    йҡҸеҗҺпјҢжөҒзЁӢиҝӣе…ҘеҶ…ж ёзә§зҡ„з»‘е®ҡдёҺжҸЎжүӢйҳ¶ж®өгҖӮеҪ“ Windows еҶ…ж ёжҺҘж”¶еҲ°еёҰжңүи°ғиҜ•ж Үеҝ—зҡ„еҲӣе»әиҜ·жұӮж—¶пјҢе®ғе№¶дёҚдјҡд»…д»…ејҖеҗҜдёҖдёӘзӢ¬з«Ӣзҡ„иҝӣзЁӢпјҢиҖҢжҳҜеңЁеҶ…ж ёж•°жҚ®з»“жһ„дёӯе»әз«Ӣиө·зӣ®ж ҮзЁӢеәҸпјҲDebuggeeпјүдёҺи°ғиҜ•еҷЁиҝӣзЁӢпјҲmsvsmonпјүд№Ӣй—ҙзҡ„зҲ¶еӯҗе…ізі»жҲ–и°ғиҜ•е…іиҒ”гҖӮиҝҷз§Қж·ұеәҰзҡ„з»‘е®ҡзЎ®дҝқдәҶзӣ®ж ҮиҝӣзЁӢдә§з”ҹзҡ„жүҖжңүејӮеёёгҖҒзәҝзЁӢеҲӣе»әжҲ–жЁЎеқ—еҠ иҪҪзӯүдәӢд»¶пјҢйғҪиғҪйҖҡиҝҮзү№е®ҡзҡ„и°ғиҜ•йҖҡйҒ“е®һж—¶еҸҚйҰҲз»ҷи°ғиҜ•еҷЁпјҢдҪҝи°ғиҜ•еҷЁиғҪеӨҹжҺҢжҸЎзӣ®ж ҮзЁӢеәҸзҡ„е®Ңж•ҙз”ҹе‘Ҫе‘ЁжңҹгҖӮ

    жңҖеҗҺжҳҜжү§иЎҢеүҚзҡ„жҢӮиө·дёҺжҺҘз®Ўйҳ¶ж®өгҖӮдёәдәҶзЎ®дҝқејҖеҸ‘дәәе‘ҳдёҚдјҡй”ҷиҝҮд»»дҪ•дёҖиЎҢд»Јз ҒпјҢзӣ®ж ҮиҝӣзЁӢеңЁеҲқе§ӢеҢ–е®ҢжҲҗеҗҺпјҢе№¶дёҚдјҡз«ӢеҚіи·іиҪ¬еҲ° main еҮҪж•°жҲ–з”ЁжҲ·е…ҘеҸЈзӮ№жү§иЎҢгҖӮзӣёеҸҚпјҢж“ҚдҪңзі»з»ҹдјҡеңЁеҠ иҪҪеҷЁпјҲLoaderпјүе®ҢжҲҗеҲқжӯҘе·ҘдҪңеҗҺпјҢиҮӘеҠЁе°Ҷзӣ®ж ҮиҝӣзЁӢзҡ„дё»зәҝзЁӢзҪ®дәҺжҢӮиө·пјҲSuspendпјүзҠ¶жҖҒгҖӮжӯӨж—¶зҡ„зӣ®ж ҮзЁӢеәҸеҰӮеҗҢдёҖеҸ°е·Із»ҸеҸ‘еҠЁдҪҶиё©дҪҸеҲ№иҪҰзҡ„жұҪиҪҰпјҢйқҷйқҷзӯүеҫ…и°ғиҜ•еҷЁзҡ„иҝӣдёҖжӯҘжҢҮд»ӨгҖӮеҸӘжңүеҪ“и°ғиҜ•еҷЁе®ҢжҲҗдәҶз¬ҰеҸ·еҠ иҪҪгҖҒж–ӯзӮ№и®ҫзҪ®зӯүеҮҶеӨҮе·ҘдҪңе№¶еҸ‘еҮә"з»§з»ӯ"е‘Ҫд»ӨеҗҺпјҢзӣ®ж ҮзЁӢеәҸжүҚдјҡзңҹжӯЈејҖе§Ӣжү§иЎҢдёҡеҠЎйҖ»иҫ‘гҖӮ

    иҝҷдёҖйғЁеҲҶжҸӯзӨәдәҶи°ғиҜ•еҷЁзңҹжӯЈ"жҺҢжҺ§"зӣ®ж ҮиҝӣзЁӢзҡ„й»‘зӣ’жңәеҲ¶гҖӮжҲ‘е°Ҷиҝҷдәӣж ёеҝғйҖ»иҫ‘ж•ҙзҗҶжҲҗдәҶжӣҙеҠ дё“дёҡдё”еҜҢжңүйҖ»иҫ‘жҖ§зҡ„ж–Үеӯ—жҸҸиҝ°пјҡ

    з¬¬дәҢйҳ¶ж®өпјҡи°ғиҜ•еҫӘзҺҜ (The Debug Loop) вҖ”вҖ” ж ёеҝғи°ғеәҰеҝғи„Ҹ¶

    и°ғиҜ•еҷЁзҡ„иҝҗиЎҢжң¬иҙЁдёҠжҳҜдёҖдёӘй«ҳж•Ҳдё”дёҘеҜҶзҡ„иҮӘеҫӘзҺҜзӣ‘еҗ¬зі»з»ҹгҖӮеҪ“и°ғиҜ•еҷЁиҝӣе…Ҙе·ҘдҪңзҠ¶жҖҒж—¶пјҢе®ғдјҡз»ҙжҢҒдёҖдёӘеёёй©»зҡ„ While LoopпјҢе…¶ж ёеҝғжһўзәҪжҳҜ WaitForDebugEvent APIгҖӮжӯӨж—¶пјҢи°ғиҜ•еҷЁдјҡиҝӣе…ҘдёҖз§Қ"й«ҳж•Ҳйҳ»еЎһ"зҠ¶жҖҒпјҢйқҷй»ҳзӯүеҫ…зӣ®ж ҮиҝӣзЁӢдёӯд»»дҪ•йЈҺеҗ№иҚүеҠЁеј•еҸ‘зҡ„дҝЎеҸ·гҖӮ

    дёҖж—Ұзӣ®ж ҮиҝӣзЁӢи§ҰеҸ‘дәҶе…ій”®дәӢд»¶вҖ”вҖ”ж— и®әжҳҜжЁЎеқ—еҠ иҪҪпјҲDLL LoadпјүгҖҒзәҝзЁӢеҲӣе»әпјҢиҝҳжҳҜејҖеҸ‘иҖ…жңҖе…іеҝғзҡ„ж–ӯзӮ№и§ҰеҸ‘вҖ”вҖ”Windows еҶ…ж ёдјҡиҮӘеҠЁд»Ӣе…ҘгҖӮеҶ…ж ёдјҡзһ¬й—ҙеҶ»з»“зӣ®ж ҮиҝӣзЁӢзҡ„жүҖжңүзәҝзЁӢпјҢе°ҶзҺ°еңәзҺҜеўғжү“еҢ…жҲҗз»“жһ„еҢ–зҡ„дәӢд»¶дҝЎжҒҜдј йҖ’з»ҷи°ғиҜ•еҷЁгҖӮи°ғиҜ•еҷЁйҡҸеҚі"иӢҸйҶ’"пјҢж №жҚ®дәӢд»¶зұ»еһӢжү§иЎҢзӣёеә”зҡ„йҖ»иҫ‘пјҡеҠ иҪҪз¬ҰеҸ·ж–Үд»¶пјҲPDBпјүд»ҘеҜ№йҪҗжәҗз ҒпјҢжҲ–жҳҜеӨ„зҗҶ EXCEPTION_BREAKPOINT ејӮеёёгҖӮжңҖеҗҺпјҢеҪ“ејҖеҸ‘иҖ…е®ҢжҲҗжҹҘзңӢе№¶жҢҮд»Өз»§з»ӯж—¶пјҢи°ғиҜ•еҷЁи°ғз”Ё ContinueDebugEventпјҢиҜ·жұӮеҶ…ж ёжҒўеӨҚзәҝзЁӢпјҢи®©зЁӢеәҸйҮҚж–°"жҙ»"иҝҮжқҘгҖӮ

    з¬¬дёүйҳ¶ж®өпјҡж–ӯзӮ№жіЁе…ҘдёҺжҢҮд»Өзә§жҺ§еҲ¶ (Control)¶

    • иҪҜд»¶ж–ӯзӮ№пјҲINT 3пјүпјҡ еҪ“дҪ еңЁд»Јз ҒиЎҢе·Ұдҫ§зӮ№дёӢзәўзӮ№ж—¶пјҢи°ғиҜ•еҷЁе®һйҷ…дёҠжҳҜеңЁзӣ®ж ҮеҶ…еӯҳзҡ„зӣёеә”ең°еқҖ"еҠЁдәҶжүӢи„ҡ"гҖӮе®ғдјҡе°ҶиҜҘдҪҚзҪ®зҡ„еҺҹжҢҮд»ӨйҰ–еӯ—иҠӮжӣҝжҚўдёә 0xCCпјҲеҚі INT 3 жҢҮд»ӨпјүгҖӮеҪ“ CPU жү§иЎҢеҲ°жӯӨеӨ„ж—¶пјҢдјҡејәеҲ¶и§ҰеҸ‘дёҖдёӘдёӯж–ӯејӮеёёпјҢдәӨз”ұи°ғиҜ•еҷЁеӨ„зҗҶгҖӮ
    • еҚ•жӯҘиҝҪиёӘпјҲSingle Steppingпјүпјҡ дёәдәҶе®һзҺ°"йҖҗиЎҢжү§иЎҢ"пјҢи°ғиҜ•еҷЁдјҡеҲ©з”Ё CPU зЎ¬д»¶еұӮйқўзҡ„йҷ·йҳұж Үеҝ—дҪҚпјҲTrap Flag, TFпјүгҖӮйҖҡиҝҮе°Ҷж Үеҝ—еҜ„еӯҳеҷЁдёӯзҡ„ TF зҪ®дёә 1пјҢCPU дјҡиҝӣе…ҘеҚ•жӯҘжЁЎејҸпјҡжҜҸжү§иЎҢе®ҢдёҖжқЎжңәеҷЁжҢҮд»ӨпјҢе°ұдјҡиҮӘеҠЁдә§з”ҹдёҖдёӘ SINGLE_STEP ејӮеёёе№¶жҢӮиө·гҖӮи°ғиҜ•еҷЁжӯЈжҳҜйҖҡиҝҮиҝҷз§Қ"жү§иЎҢдёҖжӢҚгҖҒеҒңйЎҝдёҖжӢҚ"зҡ„иҠӮеҘҸпјҢе®һзҺ°дәҶеҜ№д»Јз ҒиҝҗиЎҢз»ҶиҠӮзҡ„еҫ®и§Ӯи§ӮжөӢгҖӮ

    и°ғиҜ•д»»еҠЎз»“жқҹж—¶пјҢи°ғиҜ•еҷЁжҸҗдҫӣдәҶдёӨз§Қдјҳйӣ…зҡ„йҖҖеҮәж–№ејҸгҖӮжңҖеёёи§Ғзҡ„жҳҜе®Ңе…Ёз»ҲжӯўпјҢеҚіи°ғз”Ё TerminateProcess е№ІеҮҖеҲ©иҗҪең°з»“жқҹзӣ®ж ҮиҝӣзЁӢзҡ„з”ҹе‘Ҫе‘ЁжңҹгҖӮеҸҰдёҖз§ҚеҲҷжҳҜеҲҶзҰ»пјҲDetachпјүжЁЎејҸпјҡйҖҡиҝҮи°ғз”Ё DebugActiveProcessStopпјҢи°ғиҜ•еҷЁдјҡж’Өй”ҖжүҖжңүзҡ„еҶ…еӯҳдҝ®ж”№пјҲеҰӮжҒўеӨҚиў«жӣҝжҚўзҡ„ 0xCC еӯ—иҠӮпјүе№¶и§ЈйҷӨеҶ…ж ёз»‘е®ҡгҖӮжӯӨж—¶пјҢзӣ®ж ҮиҝӣзЁӢе°Ҷж‘Ҷи„ұжқҹзјҡпјҢжҒўеӨҚиҮізӢ¬з«ӢиҝҗиЎҢзҠ¶жҖҒпјҢеңЁдёҚе№Іжү°дёҡеҠЎйҖ»иҫ‘зҡ„еүҚжҸҗдёӢз»§з»ӯжү§иЎҢгҖӮ

    и°ғиҜ•зҡ„еҹәзҹіпјҡжһ„е»әзі»з»ҹдёҺз¬ҰеҸ·ж–Үд»¶ (The Build Process & Symbols)¶

    и°ғиҜ•е№¶йқһе§ӢдәҺ F5пјҢиҖҢжҳҜе§ӢдәҺзј–иҜ‘гҖӮжүҖд»Ҙиҝҷе°ұжҳҜдёәд»Җд№ҲиҰҒжһ„е»әDebugжЁЎејҸиҝӣиЎҢи°ғиҜ•пјҢиҰҒдёҚз„¶жІЎжңүи°ғиҜ•з¬ҰеҸ·дјҡеҫҲйә»зғҰзҡ„гҖӮ

    и°ғиҜ•зҡ„"ең°еӣҫ"дёҺ"еҗ‘еҜј"пјҡPDB дёҺзј–иҜ‘й…ҚзҪ®¶

    еҰӮжһңиҜҙдәҢиҝӣеҲ¶ж–Үд»¶жҳҜиҝ·е®«пјҢйӮЈд№Ҳ PDB (Program Database) е°ұжҳҜиҝҷд»Ҫиҝ·е®«зҡ„ең°еӣҫгҖӮе®ғе№¶дёҚжҳҜз®ҖеҚ•зҡ„иҫ…еҠ©ж–Үд»¶пјҢиҖҢжҳҜдёҖдёӘеӨҚжқӮзҡ„ж•°жҚ®еә“пјҢи®°еҪ•дәҶжңәеҷЁз Ғең°еқҖдёҺжәҗд»Јз ҒиЎҢеҸ·гҖҒеҸҳйҮҸеҗҚгҖҒзұ»еһӢе®ҡд№үд»ҘеҸҠж ҲеӣһжәҜжүҖйңҖзҡ„ FPO ж•°жҚ®гҖӮ

    еҪ“зЁӢеәҸеңЁең°еқҖ 0x00401000 еҙ©жәғж—¶пјҢи°ғиҜ•еҷЁе№¶дёҚзҹҘйҒ“иҝҷйҮҢеҸ‘з”ҹдәҶд»Җд№ҲгҖӮе®ғдјҡиҝ…йҖҹжЈҖзҙў PDB ж–Үд»¶пјҢйҖҡиҝҮжҳ е°„иЎЁеҸ‘зҺ°иҜҘең°еқҖеҜ№еә”зҡ„жҳҜ main.cpp з¬¬ 15 иЎҢгҖӮжӯЈжҳҜйҖҡиҝҮиҝҷз§Қз¬ҰеҸ·еҢ–пјҲSymbolicationпјүиҝҮзЁӢпјҢи°ғиҜ•еҷЁжүҚиғҪе°ҶеҺҹе§Ӣзҡ„еҜ„еӯҳеҷЁзҠ¶жҖҒиҪ¬еҢ–дёәејҖеҸ‘иҖ…иғҪзҗҶи§Јзҡ„д»Јз ҒдёҠдёӢж–ҮгҖӮ

    дёәдәҶзЎ®дҝқиҝҷеј ең°еӣҫзҡ„еҮҶзЎ®жҖ§пјҢзј–иҜ‘йҖүйЎ№иҮіе…ійҮҚиҰҒпјҡ

    • /Zi жҲ– /ZIпјҡејәеҲ¶з”ҹжҲҗ PDB и°ғиҜ•дҝЎжҒҜпјҢе…¶дёӯ /ZI дё“й—Ёдёә"зј–иҫ‘е№¶з»§з»ӯ"йў„з•ҷдәҶйўқеӨ–зҡ„иЎ¬еһ«з©әй—ҙгҖӮ
    • /Od (зҰҒз”ЁдјҳеҢ–)пјҡиҝҷжҳҜ Debug жЁЎејҸзҡ„зҒөйӯӮгҖӮзј–иҜ‘еҷЁеңЁдјҳеҢ–пјҲ/O2пјүж—¶дјҡдёәдәҶжҖ§иғҪйҮҚжҺ’жҢҮд»ӨжҲ–еҶ…иҒ”еҮҪж•°пјҢеҜјиҮҙдәҢиҝӣеҲ¶жөҒдёҺжәҗз ҒиЎҢеҸ·е®Ңе…Ёй”ҷдҪҚгҖӮзҰҒз”ЁдјҳеҢ–зЎ®дҝқдәҶ"жүҖи§ҒеҚіжүҖеҫ—"зҡ„и°ғиҜ•дҪ“йӘҢгҖӮ

    ж–ӯзӮ№гҖҒиҜ„дј°дёҺзғӯиЎҘдёҒ¶

    1. ж–ӯзӮ№е®һзҺ°пјҡиҪҜд»¶ vs зЎ¬д»¶¶

    • иҪҜд»¶ж–ӯзӮ№ (INT 3)пјҡеҪ“дҪ жҢүдёӢ F9пјҢи°ғиҜ•еҷЁдјҡжү§иЎҢдёҖж¬Ў"еҒ·жўҒжҚўжҹұ"гҖӮе®ғе°Ҷж–ӯзӮ№еӨ„зҡ„жҢҮд»ӨйҰ–еӯ—иҠӮжӣҝжҚўдёә 0xCCгҖӮеҪ“ CPU ж’һдёҠиҝҷдёӘеӯ—иҠӮж—¶пјҢдјҡи§ҰеҸ‘дёӯж–ӯе№¶е°ҶжҺ§еҲ¶жқғз§»дәӨз»ҷж“ҚдҪңзі»з»ҹпјҢиҝӣиҖҢйҖҡзҹҘи°ғиҜ•еҷЁгҖӮ
    • зЎ¬д»¶ж–ӯзӮ№пјҡйҖҡиҝҮ CPU зҡ„дё“з”Ёи°ғиҜ•еҜ„еӯҳеҷЁ (Dr0 - Dr7) е®һзҺ°гҖӮе®ғж— йңҖдҝ®ж”№еҶ…еӯҳпјҢйҖҡеёёз”ЁдәҺзӣ‘жҺ§еҸҳйҮҸеҸҳеҢ–пјҲж•°жҚ®ж–ӯзӮ№пјүгҖӮ

    2. иЎЁиҫҫејҸиҜ„дј° (EE)пјҡеҫ®еһӢзј–иҜ‘зі»з»ҹ¶

    еҪ“дҪ еңЁзӣ‘и§ҶзӘ—еҸЈиҫ“е…Ҙ ptr->member ж—¶пјҢVS еҶ…йғЁзҡ„иЎЁиҫҫејҸиҜ„дј°еҷЁдјҡз«ӢеҚіиЎҢеҠЁгҖӮе®ғз»“еҗҲ PDB дёӯзҡ„зұ»еһӢдҝЎжҒҜи®Ўз®—еҮәеҶ…еӯҳеҒҸз§»йҮҸпјҢзӣҙжҺҘиҜ»еҸ–зӣ®ж ҮиҝӣзЁӢзҡ„еҶ…еӯҳең°еқҖпјҢе°Ҷе…¶ж јејҸеҢ–дёәдәәзұ»еҸҜиҜ»зҡ„з»“жһ„гҖӮ

    3. зј–иҫ‘е№¶з»§з»ӯпјҡзғӯиЎҘдёҒжҠҖжңҜ¶

    иҝҷжҳҜдёҖдёӘжһҒе…·жҢ‘жҲҳжҖ§зҡ„еҠҹиғҪгҖӮеҪ“дҪ дҝ®ж”№д»Јз Ғж—¶пјҢVS дјҡеңЁеҗҺеҸ°иҝӣиЎҢеўһйҮҸзј–иҜ‘пјҢз”ҹжҲҗж–°зҡ„дәҢиҝӣеҲ¶зүҮж®өгҖӮе®ғйҖҡиҝҮ"зғӯиЎҘдёҒпјҲHot Patchingпјү"жҠҖжңҜпјҢе°ҶеҺҹеҮҪж•°зҡ„е…ҘеҸЈзӮ№дҝ®ж”№дёәдёҖдёӘи·іиҪ¬жҢҮд»ӨпјҲJMPпјүпјҢжҢҮеҗ‘ж–°з”ҹжҲҗзҡ„еҶ…еӯҳең°еқҖпјҢд»ҺиҖҢеңЁдёҚйҮҚеҗҜзЁӢеәҸзҡ„жғ…еҶөдёӢе®һзҺ°д»Јз Ғжӣҙж–°пјҲз¬”иҖ…иҜ•иҝҮпјҢеҸ‘зҺ°жңүзҡ„ж—¶еҖҷдёҚжҳҜеҫҲеҘҪз”ЁеҸҜиғҪдјҡеӨұиҙҘпјү

    еёёи§Ғй—®йўҳдёҺжҺ’жҹҘ (Troubleshooting)¶

    жіЁж„ҸеҲ°пјҢиҝҷйҮҢжҳҜдёҖдәӣеёёи§Ғзҡ„и°ғиҜ•зҡ„ж—¶еҖҷйҒҮеҲ°зҡ„й—®йўҳпјҢз¬”иҖ…жҖ»з»“д»ҘдёӢж”ҫеҲ°иҝҷйҮҢпјҡ

    1. "Breakpoint will not currently be hit" (з©әеҝғеңҶж–ӯзӮ№)пјҡ
      • еҺҹеӣ пјҡPDB дёҺжәҗд»Јз ҒдёҚеҢ№й…ҚпјҢжҲ–иҖ… PDB жңӘеҠ иҪҪгҖӮ
      • и§ЈеҶіпјҡжЈҖжҹҘ"жЁЎеқ—"зӘ—еҸЈпјҢзңӢз¬ҰеҸ·еҠ иҪҪзҠ¶жҖҒпјӣзЎ®дҝқд»Јз ҒжңӘиў«дјҳеҢ–жҺүгҖӮ
    2. еҸҳйҮҸжҳҫзӨә "Variable is optimized away"пјҡ
      • еҺҹеӣ пјҡRelease жЁЎејҸдёӢпјҢеҸҳйҮҸеҸҜиғҪиў«еӯҳж”ҫеңЁеҜ„еӯҳеҷЁдёӯеӨҚз”ЁпјҢжҲ–иҖ…зӣҙжҺҘиў«еёёйҮҸжҠҳеҸ ж¶ҲйҷӨдәҶгҖӮ
    3. е Ҷж ҲжҚҹеқҸ (Stack Corruption)пјҡ
      • и°ғиҜ•еҷЁж— жі•еӣһжәҜе Ҷж ҲгҖӮйҖҡеёёжҳҜеӣ дёәзј“еҶІеҢәжәўеҮәиҰҶзӣ–дәҶиҝ”еӣһең°еқҖгҖӮ
