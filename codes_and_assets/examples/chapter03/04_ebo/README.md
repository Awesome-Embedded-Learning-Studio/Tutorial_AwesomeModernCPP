# Chapter 3.4 - 空基类优化 (EBO) 示例

本目录包含《空基类优化（EBO）》章节的代码示例。

## 说明

空基类优化 (Empty Base Optimization) 是 C++ 编译器的优化技术，让空基类不占用派生类的空间。

## 编译方法

```bash
mkdir build && cd build
cmake ..
make
./ebo_example
```

## 关键点

1. **EBO 原理**：空基类不占用派生类空间
2. **自动启用**：现代 C++ 编译器默认启用 EBO
3. **多继承**：多个空基类可以共享同一地址空间
4. **模板设计**：EBO 是模板库的重要优化技术
5. **对齐影响**：带对齐的空基类可能影响 EBO 效果

## 环境要求

- C++20 或更高
- GCC 10+ 或 Clang 10+
