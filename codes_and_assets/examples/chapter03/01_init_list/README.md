# Chapter 3.1 - 初始化列表示例

本目录包含《初始化列表 vs 成员赋值》章节的代码示例。

## 说明

初始化列表不是语法糖，而是直接在构造阶段完成初始化，避免多余的默认初始化和赋值操作。

## 编译方法

```bash
mkdir build && cd build
cmake ..
make
./init_list_example
```

## 关键点

1. **成员赋值**：先默认初始化，再赋值（两次操作）
2. **初始化列表**：直接初始化（一次操作）
3. **必须使用初始化列表的场景**：
   - `const` 成员
   - 引用成员
   - 没有默认构造函数的成员
4. **语义完整性**：对象构造完成后处于可用状态

## 环境要求

- C++11 或更高
- GCC 4.8+ 或 Clang 3.3+
