# Chapter 6.5: Intrusive 智能指针与引用计数（非堆实现）

本目录包含侵入式智能指针的实现示例，展示如何在嵌入式环境下使用非堆分配的引用计数。

## 文件说明

| 文件 | 描述 |
|------|------|
| `intrusive_refcount.cpp` | 侵入式引用计数智能指针完整实现 |

## 关键概念

- **侵入式**: 引用计数作为对象的一部分
- **无额外堆分配**: 不需要单独的控制块
- **支持非堆对象**: 可以管理静态或内存池中的对象
- **析构分离**: 只调用析构函数，不释放内存

## 适用场景

- 内存受限的嵌入式系统
- 需要共享非堆分配对象
- 单线程环境（无需原子操作）
- 对象生命周期需要精确控制

## 编译运行

```bash
mkdir build && cd build
cmake ..
make
./intrusive_refcount
```
