# Lambda Expression Basics Examples

This directory contains examples demonstrating Lambda expression basics in modern C++.

## Examples

| Example | Description |
|---------|-------------|
| `basic_syntax.cpp` | Basic lambda syntax and structure |
| `type_deduction.cpp` | Return type deduction in lambdas |
| `algorithm_args.cpp` | Using lambdas with STL algorithms |
| `capture_basics.cpp` | Introduction to variable capture |
| `generic_lambda.cpp` | Generic lambdas with auto parameters (C++14) |
| `event_handler.cpp` | Event handling with lambdas |
| `config_builder.cpp` | Configuration builder pattern |

## Building

```bash
mkdir build && cd build
cmake ..
make
```

## Running

```bash
./bin/basic_syntax
./bin/type_deduction
./bin/algorithm_args
./bin/capture_basics
./bin/generic_lambda
./bin/event_handler
./bin/config_builder
```
